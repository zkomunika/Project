Pertidaksamaan

A. Pengertian

Pertidaksamaan adalah hubungan yang menunjukkan bahwa suatu ekspresi lebih besar, lebih kecil, lebih besar sama dengan, atau lebih kecil sama dengan ekspresi lainnya.

Simbol:

  • > : lebih besar dari.
  • < : lebih kecil dari.
  • ≥ : lebih besar atau sama dengan.
  • ≤ : lebih kecil atau sama dengan.

Cara menyelesaikan:

  • Tambahkan kedua sisi dengan bilangan yang sama.
  • Kalikan kedua sisi dengan bilangan positif.
  • Kalikan kedua sisi dengan bilangan negatif, tetapi tanda ketidaksamaan berubah.

Contoh:

2x - 1 < x + 3

2x - x < 3 + 1

x < 4

B. Jenis-Jenis Pertidaksamaan

  1. Pertidaksamaan Linear Satu Variabel
    ax + b > 0 atau ax + b < 0
    Contoh:
    3x - 5 ≤ 10
    Penyelesaian:
    3x ≤ 15 ⟹ x ≤ 5
  2. Pertidaksamaan Linear Dua Variabel
    ax + by ≥ c atau ax + by ≤ c
    Contoh:
    2x + 3y ≥ 6
  3. Pertidaksamaan Kuadrat
    ax² + bx + c > 0 atau ax² + bx + c < 0
    Contoh:
    x² - 5x + 6 > 0
    Faktorisasi:
    (x-2)(x-3) > 0
    Solusi:
    x < 2 atau x > 3
  4. Pertidaksamaan Rasional
    p(x)/q(x) > 0 atau p(x)/q(x) < 0
    Penyelesaian: Tentukan nilai nol pembilang dan penyebut.

C. Penyelesaian Grafis Pertidaksamaan

Untuk pertidaksamaan dua variabel, penyelesaian sering dilakukan dengan menggambar grafik. Langkah-langkahnya:

  1. Gambarkan garis batas dari persamaan (gunakan garis putus-putus untuk "<" atau ">" dan garis solid untuk "≤" atau "≥").
  2. Tentukan daerah solusi dengan memilih satu titik uji.
  3. Arsirlah daerah yang memenuhi pertidaksamaan.

D. Aplikasi Pertidaksamaan dalam Kehidupan Sehari-Hari

  1. Manajemen Keuangan
    Pertidaksamaan digunakan untuk membatasi pengeluaran dalam anggaran tertentu. Contoh:
    x + y ≤ 10 juta
  2. Perencanaan Produksi

    Pertidaksamaan linear sering digunakan untuk memodelkan batasan bahan baku, tenaga kerja, atau mesin.

  3. Penjadwalan dan Optimisasi

    Pertidaksamaan digunakan dalam algoritma seperti linear programming untuk memecahkan masalah optimisasi.

E. Relevansi Pertidaksamaan dengan Jurusan Informatika

  1. Algoritma dan Struktur Data
    Pertidaksamaan digunakan dalam analisis kompleksitas waktu. Misalnya:
    T(n) = O(n²)
  2. Pembelajaran Mesin (Machine Learning)

    Pertidaksamaan digunakan dalam model optimisasi seperti support vector machines (SVM) untuk menentukan hyperplane terbaik yang memisahkan data.

  3. Kriptografi

    Pertidaksamaan digunakan untuk menentukan batas keamanan algoritma berdasarkan fungsi-fungsi matematis.

  4. Grafik dan Visualisasi Data

    Pertidaksamaan digunakan untuk menentukan batas daerah plot dalam pembuatan visualisasi data.

  5. Pemrograman Linear dan Integer

    Pemrograman linear adalah metode penting dalam informatika untuk menyelesaikan masalah optimisasi dengan batasan yang direpresentasikan sebagai pertidaksamaan.

F. Simulasi Masalah Nyata dengan Pertidaksamaan

Masalah Optimalisasi Jaringan

Misalkan seorang mahasiswa jurusan informatika sedang merancang jaringan dengan batasan tertentu:

x₁ + x₂ ≤ 100 Mbps (kapasitas jaringan)
2x₁ + 3x₂ ≤ 300 Mbps (penggunaan sumber daya server)

Dengan menyelesaikan sistem pertidaksamaan tersebut, mahasiswa dapat menentukan distribusi bandwidth optimal.

Kesimpulan

Pertidaksamaan adalah alat matematika yang penting, tidak hanya dalam studi kalkulus tetapi juga dalam berbagai aplikasi praktis di jurusan informatika. Penerapan pertidaksamaan memungkinkan mahasiswa memahami dan memecahkan masalah kompleks dalam optimisasi, perancangan sistem, dan analisis data.
